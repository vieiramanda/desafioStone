# Programa Elixir - Stone
 - Na linha de comando execute: **$ javac *.java** e depois **$ java Desafio**

 - A entrada do programa será dada por 2 arquivos de texto, um contendo a lista de itens, e outro contendo a lista de emails, devendo o usuário informar os respectivos nomes dos arquivos conforme solicitado na execução do programa. Os arquivos devem estar no diretório "workspace".
 - Para realizar os testes, foram criados alguns arquivos com as listas. Caso se queira usar uma lista vazia na entrada, informar o arquivo "emptyList.txt".
 - As saídas são exibidas no console.
 - Após os resultados, o usuário deve encerrar o programa.
 - O código foi comentado no intuito de esclarecer como foi elaborada a solução.
